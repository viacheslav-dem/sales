# Учёт продаж

Внутренняя система для розничного магазина. Ведём ежедневные продажи, каталог товаров с историей цен, смотрим отчёты за месяц и выгружаем в Excel. Ничего лишнего — сделано под конкретную задачу.

## Что умеет

- **Ежедневный ввод продаж** — выбираешь дату, вбиваешь количество и цену по каждому товару. Можно ставить скидку или наценку прямо в форме.
- **Каталог товаров** — добавление, редактирование, архивирование. Цена хранится с историей: меняешь цену — старая остаётся для прошлых периодов.
- **Категории** — товары можно группировать по категориям.
- **Дашборд** — основные показатели за неделю/месяц, сравнение с прошлым периодом.
- **Месячный отчёт** — сводка по всем товарам за выбранный месяц.
- **Экспорт в Excel** — выгрузка месячного отчёта через PhpSpreadsheet.
- **Импорт из Excel** — загрузка данных из файла (формат конкретный, под наши таблицы).
- **История изменений** — лог того, что менялось в продажах.
- **Пользователи и роли** — простая авторизация, есть админ и менеджер.

## Стек

- PHP (без фреймворков, просто PDO)
- SQLite — база в одном файле, никаких серверов БД
- PhpSpreadsheet — для Excel
- Ванильный JS + CSS, без сборщиков и npm

## Установка

Нужен PHP 8.0+ и Composer.

```bash
composer install
```

Запуск локально:

```bash
php -S localhost:8000
```

Или через Laragon / OpenServer / любой другой локальный сервер — просто указать корень на папку проекта.

Первый запуск — открыть в браузере:

```
http://localhost:8000/install.php
```

Создастся база `data/sales.db`, таблицы и пользователь `admin` с паролем `admin123`. Пароль потом поменять, install.php в продакшене лучше удалить.

## Структура

Каждый PHP-файл — отдельная страница. Роутера нет, всё просто: файл = страница.

```
index.php          — редирект на дашборд или логин
dashboard.php      — главная с KPI
daily.php          — ввод продаж за день
products.php       — каталог товаров
categories.php     — управление категориями
report.php         — месячный отчёт
export.php         — выгрузка в Excel
import.php         — импорт из Excel
history.php        — история изменений
users.php          — управление пользователями
login.php/logout.php — авторизация

layout.php         — шапка, навигация, подвал
helpers.php        — общие функции (monthly_sales, форматирование)
db.php             — подключение к SQLite
config.php         — настройки (путь к БД, таймзона)
auth.php           — проверка сессии
install.php        — установка/миграция схемы

assets/            — JS, CSS, шрифты
data/              — файл базы данных (не коммитится)
```

## Заметки

- Интерфейс на русском, включая все тексты и сообщения.
- Цены можно вводить через запятую — `12,50` нормализуется в `12.50`.
- `.htaccess` закрывает доступ к `data/`, `vendor/` и служебным файлам.
- Тестов нет. Линтера нет. Это внутренний инструмент, не open-source библиотека.
